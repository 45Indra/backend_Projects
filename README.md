# Banking Management System (Java)

A simple console-based banking management system in Java.

## Features
- Create account
- Deposit
- Withdraw
- Check balance

## How to Run

1. Open a terminal in the `banking-system` directory.
2. Compile the Java files:
   ```
   javac Main.java Bank.java Account.java
   ```
3. Run the program:
   ```
   java Main
   ```
